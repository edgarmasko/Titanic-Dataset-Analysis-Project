This project analyses Titanic passenger data to uncover patterns in survival, examining how variables like age, gender, class, and family size influenced outcomes.


The source of the dataset was gathered from Kaggle, dataset link: https://www.kaggle.com/datasets/yasserh/titanic-dataset
The variables inside of the dataset are:
	•	PassengerID
	•	Survived
	•	PClass
	•	Name
	•	Sex
	•	Age
	•	SibSp
	•	Parch
	•	Ticket
	•	Fare


The tools used in this project were: Microsoft Excel, MySQL workbench & PowerBI.

The results and findings of this project are discussed in Titanic Final Report
